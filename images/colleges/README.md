# College Logos Guide

## How to Add College Logos

Place the actual college logo images in this folder with the following names:

### Required Images:

1. **iim-ahmedabad.png** - IIM Ahmedabad logo
2. **iim-bangalore.png** - IIM Bangalore logo
3. **iim-calcutta.png** - IIM Calcutta logo
4. **xlri-jamshedpur.png** - XLRI Jamshedpur logo
5. **isb-hyderabad.png** - ISB Hyderabad logo
6. **fms-delhi.png** - FMS Delhi logo
7. **spjain-mumbai.png** - SP Jain Mumbai logo
8. **mdi-gurgaon.png** - MDI Gurgaon logo

## Image Specifications:

- **Format**: PNG (with transparent background preferred) or JPG
- **Size**: 200x200 pixels to 400x400 pixels
- **Aspect Ratio**: Square (1:1) works best
- **File Size**: Keep under 100KB for fast loading

## Where to Get Logos:

1. **Official Websites**: Download from the college's official website
2. **Google Images**: Search for "[College Name] logo PNG"
3. **Wikipedia**: Many college pages have official logos
4. **College Media Kits**: Check if the college provides a media/press kit

## Fallback Behavior:

If an image is not found, the website will automatically show a placeholder with the college abbreviation. This ensures the site always looks good even without the actual logos.

## Tips:

- Use transparent PNG files for best results
- Ensure logos are high quality and clear
- Keep file names exactly as specified above (lowercase, with hyphens)
- Test the website after adding logos to ensure they display correctly
